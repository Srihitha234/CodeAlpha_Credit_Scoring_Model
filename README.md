# Credit Scoring System

## Objective
Predict an individual's creditworthiness using past financial data.

## Dataset
Home Credit Default Risk Dataset

## Features
- Data Cleaning
- Missing Value Handling
- Feature Scaling
- Logistic Regression Model
- Risk Prediction
- Streamlit Web Application

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Streamlit
- Matplotlib

## Model Performance

| Metric | Value |
|----------|----------|
| Accuracy | 68.7% |
| Precision | 15.96% |
| Recall | 67.43% |
| F1 Score | 25.81% |
| ROC-AUC | 0.744 |

## Output Categories
- 🟢 Creditworthy Customer
- 🟡 Medium Risk Customer
- 🔴 High Risk Customer
